# Lutin
Project grammar and language: Lutin analyzer and interpreter.

## Project structure

Folder  | Content
------------- | -------------
*src* | Source files
*tests* | Tests scripts
*doc* | Documentation, if done
*concep* | Conception documents etc ...
*boost* | The boost lib
*regex_example* | An example of the use of regex/boost

## Build

Release mode:
```shell
make
```
Debug mode:
```shell
make debug
```
